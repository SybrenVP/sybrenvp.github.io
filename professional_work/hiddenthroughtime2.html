Hidden Through Time 2

Hidden Through Time 2: Myths & Magic

Hidden object game sequel with 4 eras, more expansive editor mode and a complete new codebase.

I mostly worked on UGC support, but touched on all different aspects of this game.

From the start we decided to make multiple games from this single codebase. This meant a whole new approach on our development cycle and tech stack.

Released on all platforms including mobile.

Hidden Through Time 2: Discovery

Hidden object game sequel with 3 eras, more maps and reusing the existing codebase.

By reusing the existing codebase I had the opportunity to learn more about AssetBundles / Addressables in Unity. This was one of the key hard skill learning points for me.

I also used this opportunity of reusing the entire codebase to fix a bunch of issues with our development pipeline. So that we could improve our development pipeline.

Released on all platforms including mobile.
